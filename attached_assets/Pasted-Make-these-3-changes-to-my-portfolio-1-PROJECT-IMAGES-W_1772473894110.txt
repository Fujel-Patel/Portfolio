=== 2. DRAWER TRANSITION ANIMATION ===
Install: npm install framer-motion

Replace current page transitions with DRAWER effect:
When navigating from one section to another, animate like 
a drawer sliding open from bottom or side.

Use this approach with Framer Motion:

const drawerVariants = {
  hidden: { 
    clipPath: "inset(0 0 100% 0)",
    transition: { duration: 0.6, ease: [0.76, 0, 0.24, 1] }
  },
  visible: { 
    clipPath: "inset(0 0 0% 0)",
    transition: { duration: 0.6, ease: [0.76, 0, 0.24, 1] }
  },
  exit: {
    clipPath: "inset(100% 0 0% 0)",
    transition: { duration: 0.6, ease: [0.76, 0, 0.24, 1] }
  }
}

Wrap each section with:
<motion.section
  variants={drawerVariants}
  initial="hidden"
  whileInView="visible"
  exit="exit"
  viewport={{ once: false, amount: 0.1 }}
>

Also add a CURTAIN OVERLAY effect on nav click:
- When nav link clicked, a dark curtain div slides DOWN 
  then slides UP revealing new section
- Curtain color: #050510 (site background)
- Duration: 0.5s down + 0.5s up
- This gives premium "page wipe" feel

Curtain implementation:
const [curtain, setCurtain] = useState(false)

On nav click:
1. setCurtain(true) → curtain slides down (0.5s)
2. After 0.5s → scroll to section
3. After 0.8s → curtain slides up and disappears

CSS for curtain:
.curtain {
  position: fixed;
  top: 0; left: 0;
  width: 100%; height: 100%;
  background: #050510;
  z-index: 9999;
  transform: translateY(-100%);
  transition: transform 0.5s cubic-bezier(0.76, 0, 0.24, 1);
}
.curtain.active {
  transform: translateY(0%);
}
